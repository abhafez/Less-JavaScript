Stacking Cards

Cards pile up as you scroll — built with only CSS. No JavaScript needed.

Scroll down
  • Step 01

    position: sticky

    Each card is set to position: sticky with the same top value, so they all park at the same spot as you scroll past them.

  • Step 02

    sibling-index()

    The new CSS sibling-index() function gives each card a unique number. We use it to offset padding-top, making later cards appear higher and stacked.

  • Step 03

    animation-timeline: view()

    Scroll-driven animations replace JavaScript scroll listeners entirely. The browser maps your scroll position directly to the animation progress.

  • Step 04

    Depth via scale

    Cards behind the active one are scaled down slightly using sibling-index() again, creating the illusion of depth without any 3D transforms.

  • Step 05

    Zero JavaScript

    The entire effect — stacking, depth, scroll-driven animation — is pure CSS. Modern browsers ship everything needed to make this work today.

That's it. Pure CSS. 🃏
