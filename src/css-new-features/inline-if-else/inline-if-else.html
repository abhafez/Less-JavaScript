Inline if()

Stop duplicating selectors inside giant media query blocks. Branch right where the property lives. Resize the window — the box swaps padding at 600px.

I shrink under 600px

The syntax

.box {
  padding: if(
    media(max-width: 600px): 1rem,
    else: 2rem
  );
}

Browser support is early — Chrome 137+. Without it the box falls back to the default padding declared above the if().
