@scope

Two forms with the same markup. Only the one inside .login-form picks up the scoped styles — no BEM, no global leakage.

Login (scoped)

Newsletter (unscoped)

The syntax

@scope (.login-form) {
  input {
    border: 1px solid var(--accent);
    border-radius: 8px;
  }
  input:focus {
    outline: 2px solid var(--accent);
  }
}

Selectors only match descendants of .login-form. The newsletter input next to it stays untouched.
